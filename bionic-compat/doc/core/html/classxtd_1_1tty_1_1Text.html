Detailed Description

Generates ANSI codes for given text and style.

This object defines several constructors allowing to define only foreground colors, foreground and background colors, colors and attributes or only foreground color and attributes.

The method Text::str return the full ANSI espace sequence for given text and style options.

It also defines an implicit string cast operator allowing to use this object as a genuine string.

Example :

Text l_msg = Text("some text in red", color::red);
string l_val = l_msg;
