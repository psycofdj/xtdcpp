Detailed Description

template<class TClass>
class xtd::mixins::Singleton< TClass >

Singleton utility class.

Template Parameters
TClassBase class

This objects uses the CRTP pattern. To make a class T a singleton, it must inherit of Singleton<T>.

class MyHeavyObject : public xtd::mixin::Singleton<MyHeavyObject>
{
// your stuff here
};
int main(int,char**)
{
MyHeavyObject& l_instance = MyHeavyObject::get();
}
