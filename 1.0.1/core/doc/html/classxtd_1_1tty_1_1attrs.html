Detailed Description

ANSI codes for terminal output options.

This object holds one or more terminal output option.

It cannot be constructed and can only be used through pre-defined constants. Available constants are :

  • attrs::unset : Default output attribute, no effect.
  • attrs::reset : Remove all previously defined attribute.
  • attrs::dim : Reduced weight effect.
  • attrs::bold : Increased weight effect.
  • attrs::underlined : Underlined effect.
  • attrs::blink : Blinking effect.
  • attrs::reverse : brief Revered foreground and background effect
  • attrs::hidden : Hidden font effect.

Multiple constants can be combined together with operator| or operator|=. Example :

attrs l_val = attrs::unset;
l_val = attrs::bold;
l_val = attrs::bold | attrs::underlined;
l_val |= reset;

Available methods :

  • getValues : Returns internal ANSI code values.
  • isSet : Tells if at least one effect is registered.
